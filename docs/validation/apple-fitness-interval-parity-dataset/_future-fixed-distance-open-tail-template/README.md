# Future Fixed-Distance Work + Open Tail Template

Use this folder for a future workout where a distance-based Work goal completes, the runner keeps running briefly, and Apple Fitness shows both Work and Open.

## What To Drop In

- Apple Fitness screenshots: `screenshots/apple-fitness/`
- RunSignal Raw HealthKit Debug export: `exports/runsignal-diagnostics/`

No manual typing is required unless screenshots are unclear.

## Collection Checklist

1. Open Apple Fitness.
2. Open the workout.
3. Screenshot the interval rows, especially Work and Open.
4. Open RunSignal.
5. Open the same workout.
6. Open Raw HealthKit Debug.
7. Tap Share raw debug export.
8. Save the export file.
9. Drop Apple screenshots and RunSignal export into this folder.
10. Ask Codex to parse and compare.
